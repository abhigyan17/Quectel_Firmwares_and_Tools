# Quectel & Fibocom 5G Firmwares, Drivers, and Tools

This repository contains a comprehensive collection of stock firmwares, drivers, AT command manuals, and flashing tools for various Quectel and Fibocom 5G modules.

## Supported Modules

### Quectel 
* **RG520N** / **RG520N QuecOpen**
* **RM502QAE** / **RM502**
* **RM520** / **RM520 QuecOpen**
* **RM521** / **RM520&21**
* **RM530**, **RM550**, **RM551**
* **RG255C** (e.g., `RG255CGLABR01A04M4G...`)
* **RG500L** (e.g., `RG500LEUACR04A06M8G...`)

### Fibocom
* **FM160**
